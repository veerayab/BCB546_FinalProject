# BCB546_FinalProject: RNA-Seq Differential Expression
Group members: Veeraya Bamrung, Emma Tran
Original paper: https://doi.org/10.1371/journal.pone.0099625
Data GEO: GSE52778

This repository contains R scripts to replicate the RNA-seq analysis from the original paper (Himes et al., 2014). The resulting figures and tables are in the results folder.

This workflow uses RNA-seq data from airway smooth muscle cells treated with dexamethasone, a synthetic anti-inflammatory glucocorticoid commonly used in asthma therapy. Four human cell lines were exposed to 1 μM dexamethasone for 18 hours, with both treated and untreated samples collected for each line.
